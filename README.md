# Task Management Dashboard

A responsive task management dashboard built with React + Typescript + tailwind.

## Live Demo
https://task-dashboard-ten-navy.vercel.app/

## GitHub Repository
https://github.com/adityawankhede5/task-dashboard

## Features
- Add, edit, and delete tasks
- Task validation
- Filter by status
- Sort by due date
- Task summary by status
- Client-side routing (All Tasks / Completed Tasks)
- Local storage persistence
- Responsive UI

## Tech Stack
- React
- TypeScript
- Redux Toolkit
- React Router
- Tailwind
